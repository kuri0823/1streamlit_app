使用したULと説明

・selectbox
-使用目的-
利用者が表示条件を簡単に切り替えられるようにするため。
本館・分館別や自治体区分を選択することで、必要なデータだけを表示できるようにした。
-使用場所-
アプリ左側のサイドバーに配置し、「本館・分館の区分」「自治体区分」の選択に使用している。

・sidebar
-使用目的-
操作に関するUIを画面左側にまとめ、グラフ表示部分を見やすくするため。
-使用場所-
画面左側にサイドバーとして表示され、条件選択用の selectbox を配置している。

・tabs
-使用目的-
データ表示とグラフ表示を切り替えられるようにし、画面が見づらくならないようにするため。
-使用場所-
メイン画面に「データ」「グラフ」のタブを作成し、それぞれの内容を分けて表示している。

・dataframe
-使用目的-
CSVデータの中身をそのまま確認できるようにするため。
-使用場所-
「データ」タブ内で、読み込んだ図書館統計データを表示している。

・bar_chart
-使用目的-
自治体区分ごとの図書館数を比較しやすくするため。
-使用場所-
「グラフ」タブ内で、図書館数の比較を棒グラフとして表示している。

・line_chart
-使用目的-
自治体区分ごとの変化や傾向を可視化して確認するため。
-使用場所-
「グラフ」タブ内で、折れ線グラフとして表示している。

・pyplot（matplotlib）

-使用目的-
自治体別の図書館数の割合を分かりやすく表示するため。
-使用場所-
「グラフ」タブ内で、円グラフを表示するために使用している。

・title / write / caption
-使用目的-
アプリの内容やデータの出典を分かりやすく説明するため。
-使用場所-
画面上部にタイトルと説明文を表示し、画面下部にデータ出典を表示している。
