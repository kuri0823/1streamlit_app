アプリ名
図書館統計データ可視化アプリ

概要
このアプリは、e-Statに公開されている図書館統計データを用いて、日本の図書館の設置状況を可視化するアプリ。
Streamlit を用いて作成し、本館・分館別や自治体区分別に図書館数を確認できるようにした。

使用データ
出典：e-Stat
統計名：社会教育調査 図書館調査
データ形式：CSV
内容：本館・分館別、蔵書冊数別、設置者別の図書館数

使用技術
Python
Streamlit
pandas
matplotlib
GitHub
Streamlit Cloud

機能
CSVデータの読み込みと表示
本館・分館・合計データの切り替え
自治体区分（都道府県・市（区）・町・村）の選択
棒グラフによる図書館数の比較
折れ線グラフによる傾向の確認
円グラフによる構成比の可視化

工夫した点
・サイドバーを使い、条件を選択することで表示内容が変わるようにした。
・タブ機能を用いて、データ表示とグラフ表示を分け、見やすくした。
・円グラフを用いて自治体別の割合が見て分かるようにした。
